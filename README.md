# 📄 FinTrack Pro – Personal Finance Tracker

FinTrack Pro is a modern, lightweight **Personal Finance Tracker** built using pure HTML, CSS, and JavaScript. It helps users seamlessly manage their daily income and expenses with a clean, highly interactive, and intuitive interface. 

The application runs entirely in the browser and handles data persistence using the **Local Storage API**, eliminating the need for any backend server, databases, or user authentication.

🌐 **[Live Demo]((https://shivam-web-oss.github.io/fin-manager))** 

---

## ✨ Key Features

*   **Transaction Management:** Easily add and organize income and expense logs.
*   **Live Financial Summary:** Instant real-time updates for total Balance, Total Income, and Total Expense cards.
*   **Data Visualization:** Sleek cash flow visualization charts to easily monitor spending trends.
*   **Smart Filtering:** Quickly sort through transaction history (All, Income, or Expense).
*   **Multi-Currency Support:** Seamlessly switch between major currencies: USD ($), EUR (€), GBP (£), INR (₹), and JPY (¥).
*   **Personalization & Profile Settings:** Save your custom user profile name and select a preferred base currency.
*   **Persistent Dark Mode:** Modern dark theme toggle that saves your aesthetic preference across sessions.
*   **Local Storage Auto-Save:** Data automatically persists in the browser—never lose your logs on page refresh.
*   **One-Click Reset:** Clean state functionality to wipe out all stored data instantly whenever you want to start fresh.
*   **Lightning Fast:** Lightweight footprint running purely on client-side vanilla scripting.

---

## 🛠️ Tech Stack

*   **Frontend Architecture:** HTML5 (Semantic Layout)
*   **Styling & Theme Engine:** CSS3 (Custom Variables for Dark Mode, Flexbox/Grid Layouts)
*   **Core Logic:** JavaScript (ES6+ Vanilla DOM Manipulation)
*   **Data Persistence:** Web Storage API (`localStorage`)


### Prerequisites
All you need is a modern web browser (Chrome, Edge, Safari, Firefox). No installation of Node.js or npm packages is required.
